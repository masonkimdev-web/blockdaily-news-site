# 바이낸스 → 업비트 출금 방법

## 핵심 요약
- 업비트에서 **입금 주소** 확인 → 바이낸스에서 **Withdraw(출금)** → 업비트 입금 확인
- 여기서도 **네트워크(체인) 실수**가 제일 위험합니다.

---

## 1) 업비트 입금 주소 확인
1. 업비트 → 입금 → 코인 선택
2. 네트워크 확인
3. 주소 복사(필요 시 태그/메모 포함)

---

## 2) 바이낸스 출금
1. 바이낸스 → Wallet → Withdraw
2. 코인 선택
3. 업비트 주소 붙여넣기
4. 네트워크 동일하게 선택
5. 소액 테스트(권장) → 본 출금

---

## 체크리스트
- ✅ 주소/태그 정확한가?
- ✅ 네트워크 동일한가?
- ✅ 2FA 동작 확인했나?
- ✅ 소액 테스트했나?

---

## 마무리
이제 원화가 필요하면 업비트에서 매도 후 원화 출금을 진행하면 됩니다.
