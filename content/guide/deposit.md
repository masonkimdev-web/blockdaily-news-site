# 업비트 → 바이낸스 입금 방법

## 핵심 요약
- 업비트에서 코인 구매 → 바이낸스 입금 주소 확인 → 업비트 출금
- **가장 많이 터지는 문제 = 네트워크(체인) 선택 실수**

---

## 1) 어떤 코인을 보내야 하나?
초보 기준은 보통 아래 중 하나를 씁니다(상황 따라 다름).
- USDT (네트워크 선택 중요)
- XRP / XLM (전송 빠르고 수수료 낮은 편, 태그/메모 주의)

> 무엇을 고르든 **바이낸스가 지원하는 네트워크/입금 방식** 먼저 확인하세요.

---

## 2) 바이낸스에서 입금 주소 확인
1. 바이낸스 → Wallet(지갑) → Deposit(입금)
2. 코인 선택
3. **Network(네트워크)** 선택
4. 주소 복사(필요 시 Memo/Tag도 같이)

---

## 3) 업비트에서 출금
1. 업비트 → 해당 코인 → 출금
2. 바이낸스 주소 붙여넣기
3. 네트워크 동일하게 선택
4. 소액 테스트(권장) → 정상 입금 확인 → 본 전송

---

## 실수 방지 체크리스트
- ✅ 코인 종류가 동일한가?
- ✅ 네트워크가 동일한가?
- ✅ Memo/Tag가 필요한 코인인데 누락하지 않았나?
- ✅ 소액 테스트를 했나?
